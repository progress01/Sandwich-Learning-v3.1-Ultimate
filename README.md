# 🥪 Sandwich Learning v5.0

為「英文閱讀」與「情境記憶」設計的單頁網頁應用 (Single Page Application)。

沒有複雜的登入系統，不需要架設資料庫。核心概念是**「上下文三明治法 (Contextual Sandwich)」**：透過將單字挖空、搭配行內圖片 (Inline Image)，讓大腦在閱讀文章的同時，透過上下文與視覺線索來提取記憶。
> https://progress01.github.io/Sandwich-Learning-v3.1-Ultimate/

## ✨ 核心特色

* **極簡單檔**：只有一個 `index.html`，下載即用，支援離線編輯（查字典需連網）。
* **三階段學習流**：
1. **編輯 (Edit)**：貼文章、貼圖、查單字、挖空。
2. **提示 (Hint)**：根據提示詞（或上下文）嘗試回想。
3. **複習 (Test)**：全遮模式，圖片自動模糊，強迫大腦「主動回想」。


* **圖文整合**：支援 `Ctrl+V` 直接貼上截圖，圖片會插入在段落之間，建立強烈的視覺連結。
* **行動裝置優化**：針對手機/平板調整的觸控介面，支援側邊欄滑動與底部字典滑板。
* **資料自主**：資料儲存於瀏覽器 (LocalStorage)，提供 JSON 格式匯出/匯入備份。


### 1.編輯與貼圖 (Mode 1)

* **貼上文章**：直接將英文文章複製貼上到編輯區。
* **插入圖片**：
* **電腦版**：複製任何圖片，在編輯區直接按 `Ctrl+V`。
* **手機版**：點擊工具列的 `🖼️` 按鈕上傳相簿圖片。


* **挖空單字**：
* 選取文字後，點擊工具列的 `✂️` (或是按鍵盤 `M`)。
* (選填) 輸入提示詞，例如「v. 動詞」或「同義詞 happy」。



### 2.查字典

* **電腦版**：滑鼠雙擊單字。
* **手機版**：建議選取單字後自行查詢，或使用系統內建查詢功能。
* 字典面板支援真人發音與 Google 圖片搜尋。

### 3.複習模式

點擊頂部的 **「複習」** 按鈕：

* 所有挖空的單字會被隱藏。
* 所有插入的圖片會變**模糊**。
* **操作**：嘗試閱讀文章，遇到模糊圖片或色塊時，試著回想內容。真的想不起來，滑鼠移過去（或點擊）即可看答案。

## ⚠️ 資料備份 (重要)

因為本工具是純前端應用，資料是存在你的「瀏覽器暫存 (LocalStorage)」中。

* **請定期備份**：若你清除了瀏覽器快取，資料會消失。
* **如何備份**：打開側邊欄，點擊最下方的 **「📤 匯出」**，會下載一個 `.json` 檔案到你的裝置。
* **如何還原**：點擊 **「📥 匯入」** 選擇備份檔即可。

## 🛠️ 技術細節

* **架構**：純 HTML5 / CSS3 / Vanilla JavaScript (無框架)。
* **儲存**：Web LocalStorage API (搭配圖片 Canvas 壓縮)。
* **字典 API**：使用免費開源的 `dictionaryapi.dev`。
* **字體**：Inter, Noto Sans TC.
